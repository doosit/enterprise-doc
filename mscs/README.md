对于MSCS cluster的保护，只需在各个cluster node上安装英方工作机软件，并按照本文档进行配置，就可以对Cluster的共享磁盘（Shared Storage）中的内容进行保护，将共享磁盘中的数据实时备份到指定的灾备机上。

i2软件目前支持对Windows 2003 和Windows2008 Cluster的保护。
